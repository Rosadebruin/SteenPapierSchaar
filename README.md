# SteenPapierSchaar
Frontend Steen Papier Schaar opdracht Raymond

Met deze gemaakte code door mij kun je met een computer steen papier schaar spelen. Hieronder staat de uitleg hoe je het runt:

- Download allebei de files.
- Ga naar je bestanden toe en zoek het bestand "index.html".
- Dubbelklik op het bestand.
- Nu is het bestand geopend en kan je het spelletje gaan spelen.
